Seletores

HTML

Abordando o conceito e função dos seletores.


Seletor Universal

Utilizando "*" define-se todo elemento da página.
Todas as tags apresentadas como exemplo são dessa própria página.

*{
   font-size: 105%;

Altera o tamanho da fonte de toda a página.

Seletor por Tag/elemento

Define-se toda a tag para a estilização.

body{
         background-color: #131313
         color: aliceblue;

Toda a tag body apresenta o background cinza e cor de letra aliceblue.

Seletores do tipo class e id

Do tipo class inicia-se por um ".".

.exemplo{
            border: solid #212020 20px;

As tags com classe "exemplo" apresentam borda.

Do tipo id, apresenta "#" na frente.

#title{
        font-size: xx-large;

A tag h1 com id "title" apresenta fonte maior.

Seletor de atributos

Seleciona elementos a partir de atributos adicionados na forma data="exemplo". No CSS coloca-se "p[data="exemplo"]" por exemplo.
Utilizando códigos após a palavra data, obtém-se seleções especiais:
data^="exemplo" (^ ou /) - todos que o atributo começa com "exemplo" (exemplo-01, exemplo-02).
data$="exemplo" - todos que o atributo termina com "exemplo" (aa-exemplo, ab-exemplo).
data*="exemplo" - todos que o atributo tem a palavra "exemplo" em qualquer lugar(ax-exemplo-41, thg-exemplo-02).

span[data="underline"]{
        text-decoration: underline;

Os quotes com a data="underline" estão sublinhados.

Pseudo-classes

É uma função especial para um seletor.
Exemplos:
seletor:hover - adiciona estilização ao passar o cursor por cima do seletor.
seletor:link - adiciona estilização a links não visitados.
seletor:visited - adiciona estilização a links já visitados.
seletor:active - adiciona estilização ao clicar no link e segurar.

h2:hover{
        color: peru;

As tags h2 mudam de cor ao passar o cursor.

Pseudo-elementos

É uma função especial para estilizar uma parte específica do seletor.
Exemplos:
seletor::first-line - adiciona estilização na primeira linha.
seletor::after - adiciona estilização ao final.
seletor::before - adiciona estilização no início.

.exemplo::first-line{
        color: orange;

Muda a cor da primeira linha de toda tag com a classe="exemplo" para laranja.

Combinators

Explicam a relação entre dois seletores.
São quatro tipos:
"Espaço" - seletor descendente. Seleciona todos os elementos do segundo seletor que sejam descendentes do primeiro. Exemplo: todas as section dentro de um body ou todos os p dentro de um div.

body article{
        margin-bottom: 50px;

Adiciona uma margem de 50 px embaixo de todos os article dentro do body.


">" - seletor filho. Seleciona apenas os filhos ou descendentes imediatos com aquela tag. Exemplo: os p que estejam imediatamente dentro de um div.

header>h1:hover{
        color: peru;

Muda a cor das tag "h1" depois de todo header.


"+" - seletor irmão adjacente. Estiliza somente o segundo termo imediatamente após o primeiro. Exemplo: o primeiro p após todo div ou o primeiro h2 após todo h1.

p + p{
        font-size: small;
        font-style: italic;
        margin-left: 10px;

Muda a fonte de todo p adjacente a outro p e adiciona margem à esquerda.


"~" - seletor irmão em geral. Todos os elementos do segundo termo que sejam irmãos, em seguida, do primeiro. Exemplo: todos os h2 após o h1 (dentro do mesmo pai).

h2~div{
        margin-bottom: 10px;

Adiciona margem de 10px a todos os div irmão em seguida dos h2.
